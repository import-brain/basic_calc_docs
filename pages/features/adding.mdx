# Addition Tool
## Description
A tool that is probably not too necessary in here but we did include it anyway.
## Usage
**Use this tool to add 2 numbers.**\
Steps:
1. Launch the Program
<img src="https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!On%20Launch.jpg" alt="Launch The Program" width={500} height={500} />
2. Use 1 for Operation Option
<img src="https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!Select%201.jpg" alt="Select 1" width={500} height={500} />
3. Input Your First Number
<img src="https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!First%20Number.jpg" alt="Input 1st Number" width={500} height={500} />
4. Input Your Second Number
<img src="https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!Second%20Number.jpg" alt="Input 2nd Number" width={500} height={500} />
5. Get the Result Back and Answer The Next Two Prompts <img src="https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!Final%20Prompts.jpg" alt="Result And Answer" width={500} height={500} />
6. Finish, Depending on What you Answer to The Prompts, You Will Either Be Back Where You Started Or Terminated The Program
