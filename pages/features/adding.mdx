1. Launch the Program

2. Use 1 for Operation Option
<img src="/public/Addition%20Pics/!Select%201.jpg" alt="Select 1" />
3. Input Your First Number
<img src="/public/Addition%20Pics/!First%20Number.jpg" alt="Input 1st Number" />
4. Input Your Second Number
<img src="/public/Addition%20Pics/!Second%20Number.jpg" alt="Input 2nd Number" />
5. Get the Result Back and Answer The Next Two Prompts <img src="/public/Addition%20Pics/!Final%20Prompts.jpg" alt="Result And Answer" />
6. Finish, Depending on What you Answer to The Prompts, You Will Either Be Back Where You Started Or Terminated The Program
