# Addition Tool
## Description
A tool that is probably not too necessary in here but we did include it anyway.
## Usage
### Textual Steps
Use this tool to add 2 numbers.
Steps:
1. Launch the Program ![Launch The Program](https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!On%20Launch.jpg)
2. Use 1 for Operation Option ![Select 1](https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!Select%201.jpg)
3. Input Your First Number ![Input 1st Number](https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!First%20Number.jpg)
4. Input Your Second Number ![Input 2nd Number](https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!Second%20Number.jpg)
5. Get the Result Back and Answer The Next Two Prompts ![Result And Answer](https://github.com/import-brain/basic_calc_docs/blob/main/public/Addition%20Pics/!Final%20Prompts.jpg)
6. Finish, Depending on What you Answer to The Prompts, You Will Either Be Back Where You Started Or Terminated The Program
